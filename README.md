# 🎮 Number Guess

Trabalho avaliativo da disciplina de Gerência de Projetos do IFPE, desenvolvido com foco na aplicação prática das metodologias ágeis, especialmente o SCRUM, durante a produção de um jogo de adivinhação numérica.

---

## 📖 Sobre o Projeto

O **Number Guess** é um jogo simples de adivinhação de números, desenvolvido para aplicar conceitos de:

- Planejamento ágil
- Organização de backlog
- Sprints
- Trabalho colaborativo
- Desenvolvimento incremental utilizando SCRUM

O projeto também serviu como prática de desenvolvimento frontend moderno utilizando React, TypeScript e Tailwind CSS.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando:

- ⚡ Vite
- ⚛️ React
- 🟦 TypeScript
- 🎨 Tailwind CSS
- 🧩 shadcn/ui
